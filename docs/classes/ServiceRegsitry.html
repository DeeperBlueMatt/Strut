ServiceRegsitry Class

Defined in: web\scripts\vendor\amd\ServiceRegistryLite.js:5
Module: model.presentation

Allows on to register objects or functions under interface names and later retrieve them under those same interface names.

Multiple services may be registered under the same interface name. A single service may be also registered under multiple interface names.
